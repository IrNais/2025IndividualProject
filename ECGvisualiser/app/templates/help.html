ECG & Ternary Plot Visualizer - User Guide

Overview

This tool provides two main functionalities:

  • ECG Signal Visualization: Upload and visualize WFDB format ECG files
  • Ternary Plot Visualization: Upload CSV files and create ternary plots with customizable styles

ECG Signal Visualization

Supported File Formats

  • WFDB format files (.dat, .hea)
  • Multiple files can be uploaded simultaneously

Features

  • Interactive lead selection (All leads or individual leads)
  • Adjustable time window (5-60 seconds)
  • Adjustable start time
  • Real-time chart updates
  • Detailed ECG information display

How to Use

  1. Click "Choose Files" and select your WFDB files (.dat, .hea)
  2. Click "Upload WFDB" to process the files
  3. Use the controls to adjust the display:
    • Select Lead: Choose specific leads or view all leads
    • Time Window: Adjust the time range displayed (5-60 seconds)
    • Start Time: Set the starting point for the display

Ternary Plot Visualization

CSV File Requirements

  • Must contain exactly 5 columns
  • First row should be column headers
  • Required columns: "title" and "class" (case insensitive)
  • 3 additional value columns with any names

Features

  • Multiple CSV file upload support
  • Interactive data selection with checkboxes
  • Customizable point styles (color, shape, size)
  • Real-time plot updates
  • Automatic column name detection

How to Use

  1. Click "Choose Files" and select one or more CSV files
  2. Click "Upload CSV" to process the files
  3. Each file will be displayed separately with its own table and plot
  4. Use the controls to customize the visualization:
    • Checkboxes: Select/deselect data points to include in the plot
    • Select All/Deselect All: Quickly select or deselect all data points
    • Style Controls: Customize appearance for each class

Style Controls

Available Options

  • Color: Choose any color for each class
  • Symbol: Select from 10 different point shapes:
    • Circle, Square, Diamond, Cross, X
    • Star, Triangle Up, Triangle Down, Triangle Left, Triangle Right
  • Size: Adjust point size from 4 to 20 pixels

How to Customize Styles

  1. Select a class from the dropdown menu
  2. Adjust the color, symbol, and size for that class
  3. Click "Apply Styles" to update the plot
  4. Use "Reset Styles" to restore default settings

Tips and Best Practices

üí° Tips

  • For large datasets, use the checkboxes to focus on specific data points
  • Use different colors and symbols to distinguish between classes
  • The ternary plot automatically normalizes your data to percentages
  • You can upload multiple CSV files to compare different datasets

  • Value columns should contain numeric data
  • WFDB files must include both .dat and .hea files

Troubleshooting

Common Issues

  • Upload failed: Check file format and column requirements
  • No data displayed: Ensure CSV has correct headers and data
  • Plot not updating: Try clicking "Apply Styles" or check data selection
  • ECG not loading: Verify WFDB files are complete (.dat + .hea)
